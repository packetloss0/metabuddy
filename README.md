# metabuddy

I needed a tool for very specific usecase.

When exporting music from Ableton, metadata is not included. I needed a simple and fast tool to add basic metadata to file like artist name, title and cover art rather than using a massive metadata tool for it.

Every property can be locked so things such as artist name or label name can don't have to be entered each time opening a new file.

Does the job for now, gonna test it in the wild and most likely update it if needed.
